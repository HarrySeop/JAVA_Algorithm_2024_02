사용자가 원하는 개수의 정수값을 입력받아 0~9 범위의 정수값이 각각 몇 개인지 출력하는 Java 프로그램을 작성하세요.

---

입력: 정수 개수(n), n개의 정수

출력: 0~9 범위의 정수 개수

---

- 실행 예:
``` java
lab0_1:홍길동
5
0 2 4 6 6
1 0 1 0 1 0 2 0 0 0 
```

- 실행 예:
``` java
lab0_1:홍길동
20
3 4 5 6 3 4 5 -1 0 9 10 -99 99 3 4 5 6 7 8 9
1 0 0 3 3 3 2 1 1 2 
```

---
목적

- 입출력, 배열, 반복문, 조건문을 연습하여 알고리즘 구현 과제 수행을 준비한다.
